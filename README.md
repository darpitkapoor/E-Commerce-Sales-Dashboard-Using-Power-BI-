# 🛍️ E-Commerce Sales Dashboard using Power BI

## 📌 Project Overview
This project presents an interactive Power BI dashboard for analyzing e-commerce sales performance. It provides insights into sales trends, customer behavior, product performance, regional sales, and profitability through dynamic visualizations and KPIs.

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📊 Dashboard Features
- Sales KPIs
- Profit Analysis
- Customer Analysis
- Product Category Performance
- Regional Sales Analysis
- Interactive Filters & Slicers
- Dynamic Visualizations
